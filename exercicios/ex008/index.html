Formatações

Negrito/Destaque

Nessa frase temos um termo em negrito usando a tag <strong> que é semântica.

Itálico/Ênfase

Nessa frase temos um termo em itálico usando a tag <em> que é semântica.

Formas incoretas/Não mais usadas

As formas não semânticas são <b> e <i>.

Mais formatações

Texto Marcado

Podemos criar um texto marcado usando a tag <mark>.

Texto Deletado e Inserido

Podemos criar um texto deletado usando a tag <del>.
E um texto inserido usando a tag <ins>

Textos grandes e pequenos

Podemos fazer um texto grande usando <big> (Não semântica).

E podemos criar um texto pequeno usando <small> (semântica).

Texto sobrescrito e subscrito

Para inserir coisas do tipo x20/3 usando a tag <sup>. E do tipo H2O usando a tag <sub>.
